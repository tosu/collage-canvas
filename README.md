# 拼图生成器 / Collage Builder

基于纯前端的图片拼接与导出工具，支持桌面与移动端触控，提供拖拽排序、多分条长图模式、导出格式/质量选择、预览文件名标签（仅预览）、ZIP 打包下载等功能。

![界面预览](interface.png)

## 功能亮点
- 拖拽排序：桌面鼠标/触摸长按均可，跨 strip 让位预览，拖拽镜像与插入指示。
- 汉化多 strip 模式：按条切分、分隔线预览，跨条重排。
- 预览标签：可选显示文件名标签，仅预览不参与导出。
- 导出增强：PNG/JPG 选择、JPG 质量调节、尺寸与大小预览，单张下载或原生 ZIP 打包。
- 触控支持：捏合缩放、拖拽平移、双击复位；移动端提示文案。

## 使用说明
1. 打开页面后，可将图片拖入空状态卡片或点击“选择图片”上传。
2. 通过侧栏调整画布宽度、行高、间距、背景色；切换“长图发布模式”启用多条长图。
3. 拖拽图片即可排序；移动端长按进入拖拽。
4. 导出：
   - 点击“导出图片”打开设置：选择 PNG/JPG，JPG 可调质量，查看每张尺寸/大小。
   - “导出当前”逐张下载；“打包下载”生成 ZIP。
5. 复制：在非汉化模式下可复制生成图到剪贴板。

## 开发/部署
### 本地预览
```bash
# 无构建步骤，直接用静态服务器
npx serve .
# 或 python -m http.server 8080
```
访问输出的本地地址（例如 http://localhost:3000 或 8080）。

### 目录结构（核心文件）
- `index.html`：主页面与侧栏/模态结构
- `style.css` / `mobile-styles.css`：样式与响应式
- `script.js`：交互逻辑、拖拽、导出、ZIP 打包
- `layout.js`：图片排版算法
- `interface.png`：界面截图

### 部署到 Vercel（静态站）
- 根目录即静态输出目录，无需构建/输出目录配置。
- 已提供 `.vercelignore`，仅包含 README，其余 Markdown 将被忽略。
- 使用 Vercel 控制台或 `vercel --prod` 部署即可。

## 已知注意事项
- ZIP 打包为前端原生实现（store 模式），大量超大图可能占用更多内存。
- 剪贴板功能依赖浏览器权限；汉化模式下复制按钮禁用。
